## Подробное описание
```
Этот коммит добавляет Memory Bank для проектdocs: создание Memory Bank и анализ структуры проекта MaymunAIа MaymunAI и содержит подробный анализ структуры проекта.

Добавлены следующие файлы:
- productContext.md: Общее описание проекта, его цели и ключевые особенности
- activeContext.md: Текущий статус проекта, открытые вопросы и задачи
- progress.md: Отслеживание прогресса, выполненные и текущие задачи
- decisionLog.md: Архитектурные и реализационные решения с обоснованием
- systemPatterns.md: Повторяющиеся паттерны и стандарты в проекте
- improvementPlan.md: Детальный план улучшения проекта с диаграммами
- databaseStructure.md: Анализ структуры базы данных PostgreSQL

Основные результаты анализа:
1. Выявлена архитектура системы на основе RAG (Retrieval-Augmented Generation)
2. Проанализирована структура базы данных и связи между таблицами
3. Определены ключевые компоненты системы и их взаимодействие
4. Составлен план улучшений с разделением на краткосрочные, среднесрочные и долгосрочные задачи
5. Предложены оптимизации для работы с базой данных

Этот коммит закладывает основу для дальнейшей работы над проектом и улучшения его архитектуры.
```
