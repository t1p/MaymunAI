# Runbook: MTLA

## Назначение

Pack определяет routing, system prompt, presets и guardrails для MTLA.

## Обновление

1. Изменить нужные файлы в `packs/mtla/`.
2. Поднять версию в `pack.yaml`.
3. Проверить тесты маршрутизации/изоляции.
4. Задеплоить обновлённый pack как конфигурационное изменение.
